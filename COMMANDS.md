# Discord StudyBot 명령어 사용법

이 문서는 현재 봇의 버튼과 명령어 사용법을 정리한 안내서입니다. 기본 명령어 접두사는 `!` 입니다.

## 빠른 시작

1. 봇을 서버에 초대하고 필요한 권한을 부여합니다.
2. 관리자 권한이 있는 사용자가 현황판을 설치합니다.

```
!설치
```

3. 기록을 남길 텍스트 채널을 설정합니다.

```
!로그채널설정 #study-log
```

4. 주간 정산 결과를 받을 채널을 설정합니다.

```
!정산채널설정 #ranking
```

5. 음성방 입장 알림을 사용할 경우 감시할 음성채널을 등록합니다.

```
!음성알림방설정 스터디방
```

## 현황판 버튼

### ▶ 출근

공부 또는 작업 세션을 시작합니다.

- 이미 공부 중이면 다시 시작되지 않습니다.
- 휴식 중이면 먼저 복귀하거나 퇴근해야 합니다.
- 출근 후부터 실공부 시간이 계산됩니다.

### ⏸ 휴식/복귀

현재 상태에 따라 휴식을 시작하거나 공부 상태로 복귀합니다.

- 공부 중에 누르면 휴식 시작
- 휴식 중에 누르면 복귀
- 대기 중에는 사용할 수 없습니다.
- 휴식 시간은 실공부 시간에서 제외됩니다.

### ⏹ 퇴근

현재 세션을 종료하고 공부 시간을 기록합니다.

- 휴식 중 퇴근하면 현재 휴식 시간까지 반영한 뒤 종료됩니다.
- 오늘 공부 시간, 이번 주 누적 시간, 티어, 연속 출근 기록이 갱신됩니다.

### 📌 오늘 요약

오늘의 개인 기록을 보여줍니다.

표시 내용:

- 현재 상태
- 오늘 공부 시간
- 오늘 휴식 시간
- 오늘 시작 시각
- 현재 세션 시간

### 📅 주간 정보

이번 주 개인 기록을 보여줍니다.

표시 내용:

- 이번 주 총 공부 시간
- 현재 티어
- 월요일부터 일요일까지의 일별 공부 시간

### 🏅 통합 정보

장기 누적 기록을 보여줍니다.

표시 내용:

- 이번달 총 공부 시간
- 전체 누적 공부 시간
- 현재 연속 출근 기록
- 최고 연속 출근 기록
- 티어별 누적 횟수

## 일반 명령어

### !호출

특정 사람 또는 음성채널에 호출 알림을 보냅니다.

사용법:

```
!호출 @유저
!호출 유저명
!호출 유저ID
!호출 음성채널명
!호출 #음성채널
```

예시:

```
!호출 @홍길동
!호출 홍길동
!호출 스터디방
```

동작:

- 유저를 대상으로 호출하면 해당 유저를 직접 멘션합니다.
- 대상 유저가 음성채널에 있으면 현재 위치한 음성채널 이름도 함께 표시합니다.
- 음성채널을 대상으로 호출하면 해당 음성채널에 있는 사람들을 멘션합니다.
- 호출한 사용자가 음성채널에 있으면, 어느 음성채널에서 호출했는지 함께 알립니다.
- 서버 기준 재사용 대기시간은 5분입니다.

## 관리자 명령어

아래 명령어는 관리자 또는 서버 관리 권한이 있는 사용자만 사용할 수 있습니다.

### !설치

현재 채널에 스터디 현황판을 설치합니다.

사용법:

```
!설치
```

주의:

- 이미 현황판이 등록되어 있고 메시지가 살아 있으면 기존 현황판을 사용합니다.
- 봇에게 메시지 보내기, 임베드 링크, 버튼 사용 권한이 필요합니다.

### !패널복구

현재 채널의 최근 봇 메시지 중 현황판을 찾아 다시 등록합니다.

사용법:

```
!패널복구
```

사용 상황:

- 저장된 현황판 메시지 ID가 꼬였을 때
- 봇 재시작 후 기존 현황판을 다시 연결하고 싶을 때

### !로그채널설정

출근, 휴식, 복귀, 퇴근, 정산 이벤트를 기록할 텍스트 채널을 설정합니다.

사용법:

```
!로그채널설정 #채널명
!로그채널설정 채널명
!로그채널설정 채널ID
```

예시:

```
!로그채널설정 #study-log
```

### !정산채널설정

주간 정산 결과를 보낼 텍스트 채널을 설정합니다.

사용법:

```
!정산채널설정 #채널명
!정산채널설정 채널명
!정산채널설정 채널ID
```

예시:

```
!정산채널설정 #ranking
```

자동 정산 시각:

- 매주 월요일 00:00 KST
- 일요일에서 월요일로 넘어가는 자정에 지난주 기록을 닫고 새 주를 시작합니다.

### !음성알림방설정

입장 알림과 출근 깜빡 방지 알림을 사용할 음성채널을 등록합니다.

사용법:

```
!음성알림방설정 음성채널명
!음성알림방설정 #음성채널
!음성알림방설정 음성채널ID
```

예시:

```
!음성알림방설정 스터디방
```

동작:

- 등록된 음성채널에 사용자가 들어오면 텍스트 알림을 보냅니다.
- 입장 후 5분 동안 출근하지 않으면 출근 깜빡 방지 알림을 보냅니다.

### !음성알림방해제

등록된 음성 알림 대상에서 음성채널을 제거합니다.

사용법:

```
!음성알림방해제 음성채널명
!음성알림방해제 #음성채널
!음성알림방해제 음성채널ID
```

### !음성알림전체설정

서버의 모든 음성채널을 음성방 알림 대상으로 등록합니다.

사용법:

```
!음성알림전체설정
```

### !음성알림카테고리설정

특정 카테고리 안의 모든 음성채널을 알림 대상으로 등록합니다.

사용법:

```
!음성알림카테고리설정 카테고리명
!음성알림카테고리설정 카테고리ID
```

### !음성알림목록

현재 등록된 음성방 알림 대상 목록을 확인합니다.

사용법:

```
!음성알림목록
```

### !시간정정

특정 사용자의 이번 주 누적 시간을 수동으로 조정합니다.

사용법:

```
!시간정정 @유저 시간
```

예시:

```
!시간정정 @홍길동 2
!시간정정 @홍길동 -1.5
```

설명:

- 양수는 시간을 추가합니다.
- 음수는 시간을 차감합니다.
- 단위는 시간입니다.
- 차감 후 누적 시간은 0 아래로 내려가지 않습니다.

### !주간정산

주간 정산을 수동으로 실행합니다.

사용법:

```
!주간정산
```

동작:

- 현재 주간 랭킹을 출력합니다.
- 이번 주 누적 시간을 초기화합니다.
- 티어별 누적 횟수를 기록합니다.
- 진행 중인 세션은 정산 시점까지 기록한 뒤, 정산 이후 새 세션으로 이어갑니다.

주의:

- 자동 정산은 월요일 00:00 KST에 실행됩니다.
- 수동 정산은 필요할 때만 사용하는 것이 좋습니다.


### !공부데이터백업

현재 공부 데이터 파일을 채널에 첨부해 백업합니다.

사용법:

```
!공부데이터백업
```

### !공부데이터초기화

서버 설정은 유지하고 공부 기록만 초기화합니다.

유지되는 항목:

- 현황판 메시지
- 로그 채널 설정
- 정산 채널 설정
- 음성 알림방 설정

초기화되는 항목:

- 유저별 출근/휴식 상태
- 주간 누적 시간
- 날짜별 공부/휴식 기록
- 월간/누적 기록
- 연속 출근/최고 연속 출근
- 티어 횟수
- 각종 알림 상태

사용법:

```
!공부데이터초기화 확인
```

### !유저기록보기

관리자가 특정 유저의 오늘/주간/통합 기록을 한 번에 확인합니다.

사용법:

```
!유저기록보기 @유저
```

### !기록수정

특정 날짜의 공부 시간을 직접 수정합니다. 시간 단위로 입력합니다.

사용법:

```
!기록수정 @유저 YYYY-MM-DD 시간
```

예시:

```
!기록수정 @홍길동 2026-07-02 3.5
```

### !휴식수정

특정 날짜의 휴식 시간을 직접 수정합니다. 시간 단위로 입력합니다.

사용법:

```
!휴식수정 @유저 YYYY-MM-DD 시간
```

### !연속수정

현재 연속 출근 일수를 수정합니다.

사용법:

```
!연속수정 @유저 일수
```

### !최고연속수정

최고 연속 출근 일수를 수정합니다.

사용법:

```
!최고연속수정 @유저 일수
```

### !티어횟수수정

티어별 누적 횟수를 수정합니다.

사용법:

```
!티어횟수수정 @유저 티어 횟수
```

사용 가능한 티어:

- 언랭
- 브론즈
- 실버
- 골드
- 다이아
- 챌린저

### !리플레이

로그 채널의 기록을 읽어 현재 상태와 누적 기록을 복구합니다.

사용법:

```
!리플레이
```

사용 상황:

- 로컬 데이터 파일이 손상되었을 때
- 봇 상태가 실제 로그와 어긋났을 때

주의:

- 로그 채널이 먼저 설정되어 있어야 합니다.
- 마지막 주간 초기화 로그 이후의 기록을 기준으로 복구합니다.
- 로그 메시지를 읽을 수 있는 권한이 필요합니다.

## 자동 알림 기능

### 음성방 입장 알림

등록된 음성채널에 사용자가 들어오면 알림 채널에 메시지를 보냅니다.

알림 채널 우선순위:

1. 현황판 채널
2. 로그 채널
3. 정산 채널
4. 서버의 첫 번째 텍스트 채널

### 출근 깜빡 방지 알림

등록된 음성채널에 들어온 뒤 5분 동안 출근하지 않으면 알림을 보냅니다.

### 퇴근 깜빡 방지 알림

세션이 12시간 이상 이어지면 퇴근을 확인하라는 알림을 보냅니다.

### 자정 전 안내

23:50 이후에도 공부 또는 휴식 상태라면 하루가 넘어간다는 안내를 보냅니다.

### 휴식 과다 알림

휴식이 길어지면 단계별로 알림을 보냅니다.

- 30분째 휴식
- 60분째 휴식

## 티어 기준

주간 공부 시간 기준입니다.

| 티어 | 기준 |
|---|---:|
| ⚪ 언랭 | 10시간 미만 |
| 🥉 브론즈 | 10시간 이상 |
| 🥈 실버 | 20시간 이상 |
| 🥇 골드 | 30시간 이상 |
| 💎 다이아 | 40시간 이상 |
| 🏆 챌린저 | 52시간 이상 |

## 기록 처리 기준

- 기준 시간대는 KST입니다.
- 하루가 넘어가도 세션이 켜져 있으면, 이전 날짜 기록을 자동 확정하고 새 날짜 세션으로 이어갑니다.
- 주간 정산은 월요일 00:00 KST에 지난주 기록을 닫고 새 주를 시작합니다.
- 휴식 시간은 공부 시간에서 제외됩니다.

## 권한 체크

봇에 아래 권한이 있어야 정상 동작합니다.

- 메시지 보기
- 메시지 보내기
- 임베드 링크
- 메시지 기록 읽기
- 외부 이모지 사용 권한, 필요 시
- 음성 상태 확인 권한, 음성방 알림 사용 시

## 운영 팁

- 토큰 파일은 GitHub에 올리지 마세요.
- 운영 데이터 파일인 `study_data.json`은 서버에서만 관리하는 것이 좋습니다.
- 로그 채널은 일반 사용자가 수정하거나 삭제하기 어렵게 권한을 제한하는 것을 추천합니다.
